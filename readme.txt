== Description ==
FTP Upload Manager allows you to upload media files directly to an external FTP server, reducing local storage usage.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ftp-upload-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure FTP settings in the plugin settings page under 'Settings' > 'FTP Upload Manager'.
