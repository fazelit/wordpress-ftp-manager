== Description ==
این افزونه به شما امکان می‌دهد که فایل‌های رسانه‌ای را مستقیماً به سرور FTP آپلود کنید.

== Installation ==
1. افزونه را از طریق داشبورد وردپرس نصب کنید.
2. تنظیمات FTP را در بخش مدیریت پیکربندی کنید.
3. از آپلود رسانه‌ها لذت ببرید!
